Aplicacao RMI - Conta Bancaria

Arquivos:
- ContaBancaria.java     (interface remota)
- ContaBancariaImpl.java (implementacao)
- Servidor.java          (servidor)
- Cliente.java           (cliente)

Como compilar:
    javac *.java

Como executar:
    Terminal 1 (servidor):
        java Servidor

    Terminal 2 (cliente):
        java Cliente

Obs: iniciar o servidor antes do cliente.
